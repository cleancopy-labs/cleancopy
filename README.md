# 🧹 CleanCopy – Copy Without Formatting & Tracking Parameters

A lightweight, privacy-focused Google Chrome Extension built to clean your clipboard. It strips annoying rich-text formatting, hidden HTML styles, and ugly web tracking parameters (like UTM tags) instantly before you paste.

## 🚀 Key Features
- **Strip Formatting:** Copy text from any webpage as pure, plain text.
- **Remove Tracking Links:** Automatically cleans URLs by deleting tracking tokens (`utm_source`, `fbclid`, `gclid`, `igshid`, etc.).
- **Privacy First:** 100% local processing. No internet backend, no login, and zero data tracking.
- **Built-in Counter:** Watch your stats grow on a beautiful animated popup dashboard.

## 🛠️ How to Install (Developer Mode)
Since this is an open-source tool, you can load it directly into your browser for free:
1. Download this repository as a ZIP file (Click the green **Code** button -> **Download ZIP**) and extract it.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. In the top-right corner, toggle **Developer mode** to **ON**.
4. Click the **Load unpacked** button in the top-left corner.
5. Select the extracted `cleancopy` folder.
6. Pin CleanCopy 🧩 to your extension bar and enjoy a clean web experience!

## ⚙️ Supported Trackers Automatically Stripped:
- Google Analytics (`utm_*`, `gclid`, `gclsrc`)
- Facebook / Meta (`fbclid`, `fb_ref`)
- Twitter / X (`twclid`)
- TikTok (`ttclid`)
- Instagram (`igshid`)
- HubSpot, Mailchimp, Bing, and more.
